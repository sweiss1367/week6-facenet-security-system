# sample_outputs/

This folder contains pre-generated demo screenshots showing the system working.

Screenshots were produced using publicly available images from the  
**Labeled Faces in the Wild (LFW)** dataset for demonstration purposes only.

---

## What belongs here

| File | Description |
|---|---|
| `authorized_result.jpg` | Test image classified as Authorized — green bounding box |
| `unauthorized_result.jpg` | Test image classified as Unauthorized — red bounding box |
| `webcam_screenshot.jpg` | Real-time webcam mode with overlay label (optional) |

These images are manually placed here for GitHub presentation evidence.  
They are **not** auto-generated by the system (auto-generated results go to `output/`).

---

## How to generate your own

1. Add LFW demo images to `authorized_faces/` and `test_faces/`
2. Run `python main.py --build-db`
3. Run `python main.py --test-folder test_faces`
4. Copy the best results from `output/` into this folder
5. For webcam: run `python main.py --webcam` and press **S** to save a screenshot
